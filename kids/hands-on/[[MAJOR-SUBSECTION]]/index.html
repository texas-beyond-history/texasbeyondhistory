[[Notes on the HTML]]

Create a Site:

  1. Copy the contents of the _boilerplate folder into the root folder of dev.texasbeyondhistory.net
  2. Rename the folder to the name of your Exhibit converted to lowercase words separated by hyphens (e.g. “Cabeza Cooking” becomes "cabeza-cooking")
  3. For each subpage/subsection of the exhibit, duplicate the original [[barebones-index]].html file in the root of the _boilerplate folder.
  4. Rename the main page to index.html
  5. Rename other files to meaningful shorthands of the title of the page (e.g. “About” becomes about.html; “Roots and Fish of Coastal Foodways” becomes coastal.html)
  6. Replace all content in [[BRACKETS]] with content that matches its purpose (see below for details)
  7. Copy sections of html from this page to create the visual effects shown here

[[Section Three]] Lists

Bulleted unordered list (ul class="bulletted")

  • An Internal Link points to a page on texasbeyonhistory.net so we open the page in the same browser tab.
  • An External Link is a link to a site other than texasbeyondhistory.net. It's common to have links to external sites in open a new tab. To accomplish this, add 'target="_blank"' to your link.

Numeric ordered list (ol class="numeric")

  1. Use this kind of markup when you want a numbered list of things.
  2. Each <li> </li> will be preceded by an integer in sequence.
